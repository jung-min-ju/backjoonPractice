## 1. 문제요약
- N개(짝수)의 사람이 있고. 이를 2팀으로 나눈다.
- N명의 사람은 1~N까지의 번호를 받았다. 그리고 서로가 같은 팀이 되었을 때의 능력치는 표에서 알 수 있다.
- 2팀의 능력치의 차가 최소가 되도록 하라.
- S12와 S21은 값은 같지만 2번 더해야 한다.

## 2. 입출력
### [입력]
- N(4 ≤ N ≤ 20, N은 짝수)
- S내 값은 1이상 100 이하

## 3. 알고리즘
- 백트래킹

[전체 로직]
- main 함수에서, N개의 사람에 대해서, 1부터 N까지 반복문을 돌리고, 본인 포함/미포함 재귀 함수 실행
  - for문의 시작은 depth부터 시작. (앞에서 본인 미포함 시킨 게 경우의 수기에, 앞의 사람 포함시키면 안됨)
  - 만약 depth가 /2가 되었다면, 현재 인자로 들어온 사람들의 경우의 수에 대한 능력치 조사해 반환 
- main 함수 내에서, 반환된 2가지 재귀함수에 대한 차를 계산하고, 전역 변수에 새롭게 갱신하기.

[능력치 조사 함수]
- 이중 for문으로 돌려야 할 듯
