## 1.문제 요약
- N * N 보드가 존재한다. 
- 뱀은 매 초마다 이동한다.
  - 시작칸은 맨위맨좌측 1,1. 오른쪽을 향하며 매초 1칸씩 늘어난다.
  - 특정 초에 왼쪽/오른쪽으로 회전하는 명령어가 주어진다.
  - 이동한 칸에 사과가 없으면, 꼬리칸을 비운다.(뱀 길이 변화x)
  - 이동한 칸에 사과가 있으면, 꼬리칸유지+머리1칸 늘리기(뱀 길이 변화0)
- 뱀이 자신의 몸통이나 벽에 닿으면 게임이 종료됨.

## 2. 입출력
[입력]
- N(2<=N<=100)
- 사과 개수 K((0<=K<=100))
- 사과 위치 행,열
- 방향 변환 횟수
- 누적 초 X(10,000 이하), 왼쪽(L)/오른쪽(D)

[출력]
- 게임이 몇 초에 끝나는지 출력

## 3. 알고리즘
- 보드 : int board[N+2][N+2] 준비 (벽=-1, 사과=-2, 뱀 움직임 : 1초~ )
