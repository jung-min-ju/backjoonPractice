## 1. 문제요약
- 수열이 주어지고, 각 수 사이에는 4가지의 연산자를 끼워넣을 수 있다.
- 연산자 우선 순위는 무시하고 앞에서부터만 진행한다.
- 수의 순서를 바꿀 수 없다.
- 만들 수 있는 결과가 최대, 최소인 것을 구하라

## 2. 입출력
### [입력]
- N(2<=N<=11)
- 수열 N개
- 덧셈, 뺄셈, 곱셈, 나눗셈의 개수 주어짐

### [출력]
- 만들 수 있는 식의 최댓값, 최솟값 출력
- 항상 결과는 +-10억임


## 3. 알고리즘
- 연산자 들어갈 수 있는 자리 9개 -> 최대 경우의 수 9! -> 70만
- 브루트포스 + 백트래킹
- 재귀로 조합 만들기 + MAX, MIN 값 동시에 찾기
