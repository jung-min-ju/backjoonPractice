## 1. 문제 요약
- N개의 정수로 이루어진 수열이 있을 때, 크기가 양수인 부분수열이 있다.
- 해당 수열의 원소의 합이 S가 되는 경우의 수를 구하라.

## 2. 입출력
[입력]
- N (1<=N<=20)
- |S| (|S| <= 1,000,000)
- 정수 n개가 주어짐. 해당 정수는 절대값 100,000을 넘지 않음.

[출력]
- 합이 s가 되는 부분 수열의 개수 출력

## 3. 테스트 케이스
5 0
-7 -3 -2 5 8

## 4. 알고리즘
- 오름차순 정렬
- 맨 뒤 원소에서 출발해 앞의 원소를 포함한 경우, 포함하지 않은 경우 재귀 진행. 
  [재귀 멈춤 조건]
    - 더 이상 합칠 원소가 없을 때
    - S 보다 작아질 때
- 경우의 수 표시할 배열 생성. 재귀를 돌며 합이 S라면, 현재 인덱스의 경우의 수 카운트
- 최종 경우의 수 다 더해줌.
