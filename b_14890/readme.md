## 1. 문제요약
- N*N 미로에 각 칸에는 높이가 적혀있음
- 높이 1, 가로 L개의 계단은, 각 칸의 높이 차가 1이고, 낮은 쪽에 L개의 평평한 지형이 있어야 놓을 수 있다.
- 한 행/한 열을 지나갈 수 있는 길의 개수를 구하라.

## 2. 입출력
### [입력]
- N(2<=N<=100), L(1<=L<=N)
- 각 칸의 높이는 10보다 작거나 같은 자연수
- 시간 : 2초

## 3. 알고리즘
- 각 행.열에 해당하는 숫자들을 [N*2][N] 배열에 전체 저장
- 해당 배열의 행을 전체적으로 돌며 로직 수행
  - 단차 발생 시, 차이가 1 이상이면 바로 중단
  - 올바른 벽(동일하거나, 경사로를 놓을 수 있는 벽)일 시 cnt++
  - 이전과 동일한 높이의 벽이 온다면, stack++진행.
  - 이전보다 큰 벽이 온다면, 현재까지 진행된 stack이 L보다 크거나 같은지 확인
  - 이전보다 작은 벽인 온다면, 앞으로 진행될 stack이 L보다 크거나 같은지 확인
    - 미래에서 확인해야 하기에, boolean 변수 활용
  - 행 탐색이 끝났을 시, boolean 변수가 false이며 정상 벽 카운팅 cnt가 N과 같을 시, answer++
