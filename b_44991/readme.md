## 문제 설명
1. 직사각형 모양의 방을 로봇이 청소한다.
2. 방에는 깨끗한 방, 더러운 방, 가구가 있는 방의 종류가 있다.
3. 로봇은 인접한 칸으로 이동할 수 있고, 같은 칸을 방문할 수 있다.
4. 더러운 칸을 모두 깨끗한 칸으로 만들기 위한 최소 횟수를 구하라.

## 입출력
### [입력]
- 입력은 여러개의 테스트 케이스로 이루어진다.
- 방 가로 크기 w, 세로 크기 (1 ≤ w, h ≤ 20)
- 방의 정보가 주어진다.
    - .: 깨끗한 칸
    - *: 더러운 칸 (최대 10칸)
    - x: 가구 (로봇 이동 불가)
    - o: 로봇 청소기의 시작 위치
- 로봇 청소기의 개수는 1개이다.
- 입력이 모두 끝나면 0 0 이라는 w, h 값이 주어진다.

### [출력]
- 더러운 칸을 모두 청소하기 위해 로봇이 이동해야 하는 최솟값
- 방문 불가 더러운 칸 존재 시 -1 출력

## 알고리즘
- 입력 받을 때 더러운 칸의 총 개수 세기
- bfs로 map[x][y][방향값] 을 visit 체크 하며 돌기
- 더러운 칸 모두 발견 시 움직인 수 바로 출력
- 큐를 모두 돌았음에도 더러운 칸을 모두 발견하지 못했다면, -1 출력
