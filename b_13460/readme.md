## 1. 문제설명
- 구슬 탈출 : 직사각형 보드에 빨간,파란 구슬을 넣고 빨간 구슬을 구멍으로 빼내는 게임
- N*M 직사각형 보드가 있고, 가장 바깥쪽 행과 열은 막혀있다. 
- 파란 구슬이 구멍으로 나오면 안된다. 보드를 4가지 방향으로 돌려 구슬을 움직인다.
- 기울이는 동작을 그만하는 것은 더 이상 구슬이 움직이지 않을 때 까지?

## 2. 입출력
### 입력
- N, M (3<=N,M<=10)
- 보드모양 주어짐
  - '.' : 빈칸
  - '#' : 벽
  - 'o' : 구멍
  - 'R,B' : 구슬 위치

### 출력
- 빨간 구슬만 구멍으로 나오는 최소 움직임 출력
- 10번 이하로 빨간 구슬을 빼낼 수 없다면 -1 출력

### 3. 알고리즘
- 구슬들은 한 번의 기울기에서 갈 수 있는 끝까지 간다.
  - 그 사이 구슬에 빠질 수 있다.
- 같은 행이나 열에서 굴러갈 때, 각 구슬의 위치도 파악해 옮겨야 한다.
