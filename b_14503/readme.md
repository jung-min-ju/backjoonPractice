## 1. 문제 요약
- N*M 방. 각 칸은 벽1, 빈칸0. 
- 청소기의 방향은 동서남북 중 하나.
- 빈 칸은 전부 청소 X 상태
- 로봇 청소기 동작 
  - 현재 칸이 청소 X라면 청소함
  - 주변 4칸에서 청소 필요한 곳 없으면, 방향 유지하며 1칸 후진
    - 뒤쪽 칸이 벽이라 후진이 불가능할 시, 작동 멈춤
  - 주변 4칸에서 청소 필요한 곳 발견
    - 반시계 방향으로 90도 회전. 해당 방향 칸 청소 X라면 전진

## 2. 입출력
### [입력]
- N, M (3<=N,M<=50)
- 로봇 청소기 좌표 r,c, 방향 d
  - 북동남서 / 0123

### [출력]
- 작동을 멈출 때 까지 청소한 칸의 개수

## 3. 알고리즘
- 현재 방향에서 **"반시계로"** 90도 회전시키고 시작. 해당 방향 빈 칸인지 확인
  - 빈 칸이라면 재귀 함수 실행.
  - 빈 칸이 아닐 시 다음 방향으로 넘어감.
- 모든 방향 탐색 끝날을 시, 후진
  - 후진 방향이 벽이라면, 전역 변수 boolean 활용해 모든 재귀함수 종료.
