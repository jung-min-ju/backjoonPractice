## 1. 문제 요약
1. 중위 표기식을 후위 표기식으로 변환하자.

### [예시]
1. a+b*c 
2. (a+(b*c)) => 우선순위가 가장 높은 *를 괄호 밖으로 꺼낸다.
3. (a+bc*) => 그 다음 우선순위인 +를 괄호 밖으로 꺼낸다.
4. abc*+

## 2. 입출력
[입력]
*  -AB 처럼, -가 앞에 오는 수식은 없다.
* AB 처럼 *가 숨겨진 수식은 없다.
* 표기식은 알파벳 대문자와 +, -, *, /, (, ) 로만 이루어진다.
* 입력값 길이는 100을 넘지 않는다.

[출력]
- 후위 표기식

## 3. 알고리즘
- 맨앞과 맨뒤 괄호 추가
- *과 / 찾아서 괄호 추가
  - 만약 피연산자 자리에 ( 있다면, 오른쪽으로 가서 ) 나온 첫 인덱스 +1에 ) 추가
  - 만약 피연산자 자리에 ) 있다면, 왼쪽으로 가서 ( 나온 첫 인덱스 -1 에 ( 추가
- +와 - 찾아서 괄호 추가
  - 만약 피연산자 자리에 ( 있다면, 오른쪽으로 가서 ) 나온 첫 인덱스 +1에 ) 추가 
  - 만약 피연산자 자리에 ) 있다면, 왼쪽으로 가서 ( 나온 첫 인덱스 -1 에 ( 추가

- stack 에 담아 편하게 결과물 출력
