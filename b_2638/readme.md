### 1. 문제요약
- N*M 모눈종이 위 특정 구역들에 치즈가 있다.
- 가장자리에는 치즈가 놓이지 않는다.
- 외부 공기칸과 2칸 이상 맞닿아 있다면 해당 치즈 격자는 한 시간 후 사라진다.
- 치즈 내부 공간은 외부 공기가 아니다.  
  → 외부 공기와 연결될 시 외부 공기로 간주된다.
- 모든 치즈가 녹는 시간을 구하는 문제.

---

### 2. 입출력
[입력]
- N, M (5 ≤ N, M ≤ 100)
- 격자 정보:
  - 치즈 = 1
  - 공기 = 0

[출력]
- 모든 치즈가 녹아 없어지는 시간

---

### 3. 알고리즘

[초기 세팅]
- 1로 표시된 구역(치즈)을 모두 큐에 집어넣고 전체 치즈 개수를 기록한다.
- BFS를 사용하여 외부 공기(0)를 모두 탐색해 2로 표시한다.
  - 이제 내부 공기는 0, 외부 공기는 2로 구분된다.

[치즈 녹이기]
- 큐에서 치즈를 하나씩 꺼내며 “녹을 수 있는지” 확인한다.
  - 외부 공기(2)와 맞닿은 면이 2개 이상이면 녹일 수 있는 치즈로 판단한다.
  - 녹은 치즈는 0으로 변경하고, 녹은 개수를 카운트한다.
  - 만약 녹은 치즈 인접 영역이 내부 공기라면 BFS로 탐색해 외부 공기(2)로 변경한다.
- 한 시간이 경과할 때마다 time을 1 증가시킨다.
- 전체 치즈 개수에서 녹은 치즈 개수를 뺀다.
- 큐가 빌 때까지 위 과정을 반복한다.

[치즈 녹이는 기준]
- 상하좌우 중 외부 공기(2)와 맞닿은 칸이 2개 이상인 경우

[종료 조건]
- 더 이상 녹을 치즈가 없을 때 종료하며, 현재 time 값을 출력한다.
