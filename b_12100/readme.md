## 1. 문제 요약
[2048 게임 규칙]
- 블록은 4방향으로 자기가 갈 수 있는 만큼 끝까지 움직인다.
  - 해당 방향으로 움직였을 때, 본인과 같은 수를 만나면 두 수는 합쳐져 하나의 블록이 된다.
  - 한 번의 이동에서 합쳐진 블록은 또 합쳐질 수 없다.
  - 마지막으로, 똑같은 수가 세 개가 있는 경우에는 이동하려고 하는 쪽의 칸이 먼저 합쳐진다. 

## 2. 입출력
[입력]
- 보드의 크기는 최대 N*N이다. (1<=N<=20)
- 블록의 초기값 입력. 적어도 1개 이상 주어짐

[출력]
- 5번 이동시켜 얻을 수 있는 가장 큰 블록을 구하여라.
